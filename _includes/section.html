Navbar

The navbar by default is fixed to the top of the page. The navbar as uses two different colors primary-code and grad-color that can be found in the _base-styles.scss. It can easily be changed throught the grid component.

Container

Containers are the main building block of the layout in Bento. Containers by default center themselves on the page and are set to a max of 960px. This can be easily changed in the base/_base-style.scss and all the components will change to reflect the bigger size. To use a container just add container as a class name.

container

Dark Section

Dark Section's are full width bands that can be used to divide up the page. To use a Dark Section add the .darkSection as a class name. Make sure that the element also does not have .container as well.
