Grid

Bento's grid makes use of flexboxes. By default the flex container is set to center items vertically and space-between items. To use the grid give a parent container the class of .grid. Then give the children of the .grid a class name of .one through .twelve .

Center modifier

By adding a .center to your grid's class name it will center any grid item inside of the grid. This will override the .start modifier.

You can make items in the grid stack instead of going horizontally by add the .column class name to the grid. This modifier does not override the .start modifier
